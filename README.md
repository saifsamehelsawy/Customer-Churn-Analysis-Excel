# 📊 Customer Churn Analysis Dashboard (Excel)

## 📌 Project Overview

This project was completed as part of my **Data Analysis (Excel) Internship** at **Codveda Technologies**.

The objective of this project is to analyze customer churn data, identify key factors affecting customer retention, and build an interactive Excel dashboard to support data-driven decision-making.

---

## 🎯 Project Objectives

- Clean and prepare the dataset for analysis.
- Analyze customer churn behavior.
- Create Pivot Tables and Pivot Charts.
- Build an interactive Excel Dashboard.
- Generate business insights from the data.

---

## 🛠️ Tools & Technologies

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Tables
- Data Cleaning Techniques

---

## 📂 Dataset

The dataset contains customer information, including:

- Customer ID
- State
- Account Length
- International Plan
- Voice Mail Plan
- Customer Service Calls
- Churn Status

---

## 📈 Dashboard Features

The interactive dashboard includes:

- ✅ Total Customers KPI
- ✅ Churn Customers KPI
- ✅ Active Customers KPI
- ✅ Churn Rate KPI
- ✅ Customer Churn Distribution
- ✅ Churn by International Plan
- ✅ Churn by Voice Mail Plan
- ✅ Churn by Customer Service Calls
- ✅ Churn by State
- ✅ Interactive Slicers

---

## 🧹 Data Cleaning

The following preprocessing steps were performed:

- Removed duplicate records.
- Checked for missing values.
- Standardized data formatting.
- Converted the dataset into an Excel Table.
- Verified data consistency.

---

## 📊 Analysis Performed

- Customer Churn Distribution
- International Plan Analysis
- Voice Mail Plan Analysis
- Customer Service Calls Analysis
- State-wise Churn Analysis

---

## 💡 Key Insights

- The overall customer churn rate is **14.37%**.
- Customers with an International Plan tend to have a higher churn rate.
- Higher customer service call frequency is associated with increased churn.
- Churn distribution varies across different states.
- Voice Mail Plan customers generally show better retention.
